## 内容

"ガベージコレクションのアルゴリズムと実装" を輪講します。
内容を全て理解するまで帰宅することのできない、ガチムチの輪講です。

* 輪講
* 著者による基調講演
* 著者への質疑応答
* 他、GCや言語処理系開発に関するプレゼン
